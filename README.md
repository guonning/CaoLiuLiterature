# CaoLiuLiterature
##草榴社区文学应用
* 自动抓取草榴文学板块帖子，并缓存本地
* 只抓取正文内容，采用对手机友好的版面，自适应手机屏幕分辨率
* 只读取楼主，自动区分是否完结
* 目前只抓取文学版块

##CaoLiuLiterature App
* Auto fetch CaoLiu Literature in the post, and cached locally
* Only crawl text content, using the phone-friendly layout, adaptive cell phone screen resolution
* Read only the Thread starter, automatically distinguish whether the end
* Currently only crawl Literature Forum
